# 欢迎来到第 1 天

这是 Rust 基础的第一天。今天我们将覆盖很多内容：

- 基本的 Rust 语法：变量、标量和复合类型、枚举、结构体、
  引用、函数和方法。
- 类型和类型推断。
- 控制流构造：循环、条件语句等。
- 用户定义类型：结构体和枚举。
- 模式匹配：解构枚举、结构体和数组。
