# 块和作用域

## 块

Rust 中的一个块包含了一系列表达式，由大括号 `{}` 包围。
每个块都有一个值和一个类型，这些都是块中最后一个表达式的值和类型：

```rust,editable
fn main() {
    let z = 13;
    let x = {
        let y = 10;
        println!("y: {y}");
        z - y
    };
    println!("x: {x}");
}
```

如果最后一个表达式以 `;` 结尾，那么结果值和类型就是 `()`。

<details>

- 你可以通过更改块中的最后一行来展示块的值是如何变化的。例如，添加/删除一个分号或使用 `return`。

</details>
