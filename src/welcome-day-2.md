# 欢迎来到第 2 天

现在我们已经了解了相当多的 Rust，今天我们将重点讨论 Rust 的类型系统：

- 模式匹配：从结构中提取数据。
- 方法：将函数与类型关联。
- 特质：多个类型间共享的行为。
- 泛型：在其他类型上参数化类型。
- 标准库类型和特质：Rust 丰富标准库的一瞥。
