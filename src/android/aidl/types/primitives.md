# 基本类型

基本类型在（大部分情况下）能够直观对应：

| AIDL 类型 | Rust 类型 | 备注                                  |
| --------- | --------- | ------------------------------------- |
| `boolean` | `bool`    |                                       |
| `byte`    | `i8`      | 注意字节是有符号的。                   |
| `char`    | `u16`     | 注意这里使用的是 `u16`，而非 `u32`。   |
| `int`     | `i32`     |                                       |
| `long`    | `i64`     |                                       |
| `float`   | `f32`     |                                       |
| `double`  | `f64`     |                                       |
| `String`  | `String`  |                                       |
